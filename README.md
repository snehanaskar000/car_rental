# Car Rental Management System

## Overview

Car Rental Management System is a modern web-based application that allows users to browse available vehicles, make car reservations, manage bookings, and process rental payments efficiently.

The project was developed to simulate a real-world car rental platform and provide practical experience in full-stack web development using PHP, MySQL, JavaScript, and modern UI design principles.

## Features

### User Authentication

* User Registration
* Secure Login System
* Session Management
* Profile Management

### Car Management

* Dynamic Car Listing
